# naisdevice up

Should preferably be run on a clean system.

![](naisdevice up.app/Contents/Resources/naisdevice-up.png)


### Usage: 
* Download .app bundle
* Run `sudo xattr -d -r com.apple.qurantine /path/to/naisdevice\ up.app`
* Doubleclick should do the trick
